# Final Challenge Bootcamp Fullstack Portales

This is a solution to the final challenge from Sophos Solutions and Ruta N.

## Overview

### The challenge

Create a Web application that will be used by employees of a financial institution. That application will allow the administration of the clients, which includes customer registration, updating their data, and deleting them. Also, it will allow you to create financial products for your clients. Finally, the financial institutions may make transactional movements and issue statements of accounts in relation to these products.
